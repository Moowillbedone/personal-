# IA 구조도 - AI 방송 썸네일 텍스트 자동 생성

## 전체 Information Architecture

```
셀러(그리퍼) App
│
└── 라이브 시작
    │
    └── 방송 설정 (메인 화면)
        │
        ├── 1. 커버 이미지 영역
        │   ├── 커버 이미지 업로드 (필수)
        │   │   ├── 직접 업로드 이미지
        │   │   └── 판매 상품 이미지 자동 불러오기 (이미지 미업로드 시)
        │   │
        │   ├── [✦ AI 커버 생성] 버튼
        │   │   ├── 생성 조건 검증
        │   │   │   ├── (필수) 커버 이미지 존재 여부
        │   │   │   ├── (필수) 방송 제목 작성 여부
        │   │   │   ├── (옵션) 판매 상품 선택 여부
        │   │   │   └── (옵션) 강조할 혜택 입력 여부
        │   │   │
        │   │   ├── 텍스트 데이터 매핑 (백그라운드 로직)
        │   │   │   ├── Line 1 - 상단 텍스트 (최대 8자)
        │   │   │   │   ├── 1순위: 대표 상품명 첫 8자 (단어 단위 컷)
        │   │   │   │   └── 2순위: 방송 제목 첫 단어 (브랜드+카테고리)
        │   │   │   │
        │   │   │   └── Line 2 - 하단 텍스트 (최대 8자)
        │   │   │       ├── 1순위: 방송 제목 내 할인율 (예: ~50% 할인)
        │   │   │       ├── 2순위: 강조할 혜택 텍스트 (예: 무료배송 혜택)
        │   │   │       └── 3순위: 기본 문구 랜덤 (라이브 특가, 단독 특가 등)
        │   │   │
        │   │   └── 텍스트 렌더링 엔진 (디자인 조합)
        │   │       ├── 폰트 2종: Pretendard / Cafe24
        │   │       ├── 하이라이트 색상 3종 (랜덤)
        │   │       ├── 블록 배경색 3종 (랜덤)
        │   │       ├── 레이아웃 4종
        │   │       │   ├── 기본 + 강조
        │   │       │   ├── 강조 + 기본
        │   │       │   ├── 기본 + 하이라이트
        │   │       │   └── 강조 + 강조
        │   │       ├── Text-Shadow 적용
        │   │       ├── Stroke 속성 적용
        │   │       └── 이미지 중앙 정렬
        │   │
        │   ├── [✦ AI 커버 제거] 버튼 (생성 완료 후 전환)
        │   │   └── 원본 이미지 상태로 복구
        │   │
        │   └── 결과물 렌더링 (커버 썸네일 박스 내 즉시 노출)
        │
        ├── 2. 방송 제목 입력 (필수)
        │
        ├── 3. 판매 상품 선택 (옵션)
        │   └── 상품 D&D → 대표 상품 순서 결정
        │
        ├── 4. 강조할 혜택 입력 (옵션)
        │
        └── 5. AI 방송커버 툴팁 (ℹ️)
            └── Bottom-up 팝업
                ├── 타이틀: "AI 방송커버가 뭐에요?"
                ├── 컨텐츠: 샘플 이미지 3장 제공
                └── 서브 텍스트: 기능 설명
```

## Phase 2 확장 영역 (선택적)

```
방송 설정 (확장)
│
├── 데이터 변경 시 동기화 정책
│   ├── A안: 자동 갱신 (리소스 높음)
│   ├── B안: 자동 해제 + 토스트 알림 (권장)
│   └── C안: 경고 팝업 후 사용자 선택
│
├── 어뷰징 방지
│   ├── 생성 중 버튼 Disable
│   └── 1분당 최대 생성 횟수 제한 (예: 10회)
│
├── 금칙어 정책
│   └── AI 커버 생성 시 금칙어 API 검증
│       └── 적발 시 생성 차단 + 토스트 메시지
│
└── 원본 이미지 규격 예외 처리
    └── 9:16 비율 Crop 강제 적용
```
