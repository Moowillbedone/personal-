# 유저 플로우 - AI 방송 썸네일 텍스트 자동 생성

## 메인 유저 플로우 (AI 커버 생성)

```mermaid
flowchart TD
    A([셀러 - 라이브 방송 시작]) --> B[방송 설정 메인 화면 진입]

    B --> C{커버 이미지가<br/>업로드 되어 있는가?}
    C -->|No| C1[커버 이미지 업로드]
    C1 --> D
    C -->|Yes| D{방송 제목이<br/>작성되어 있는가?}

    D -->|No| D1[방송 제목 작성]
    D1 --> E
    D -->|Yes| E["[✦ AI 커버 생성] 버튼 탭"]

    E --> F{생성 조건<br/>충족 여부 검증}

    F -->|미충족| G["토스트 메시지 노출<br/>(이미지 미업로드 / 제목 미작성)"]
    G --> B

    F -->|충족| H[텍스트 데이터 매핑 시작<br/>- 백그라운드 로직 -]

    H --> I[Line 1: 상단 텍스트 추출]
    I --> I1{판매 상품이<br/>선택되어 있는가?}
    I1 -->|Yes| I2["1순위: 대표 상품명 첫 8자 추출<br/>(단어 단위 컷오프)"]
    I1 -->|No| I3["2순위: 방송 제목 첫 단어 추출<br/>(브랜드+카테고리)"]

    I2 --> J[Line 2: 하단 텍스트 추출]
    I3 --> J

    J --> J1{방송 제목에<br/>할인율이 있는가?}
    J1 -->|Yes| J2["1순위: 할인율 노출<br/>(예: ~50% 할인)"]
    J1 -->|No| J3{"'강조할 혜택'이<br/>선택되어 있는가?"}
    J3 -->|Yes| J4["2순위: 혜택 텍스트 노출<br/>(예: 무료배송 혜택)"]
    J3 -->|No| J5["3순위: 기본 문구 랜덤 노출<br/>(라이브 특가, 단독 특가 등)"]

    J2 --> K[텍스트 렌더링 엔진]
    J4 --> K
    J5 --> K

    K --> K1["디자인 조합 랜덤 생성<br/>• 폰트 2종 중 선택<br/>• 하이라이트 색상 3종 중 랜덤<br/>• 블록 배경색 3종 중 랜덤<br/>• 레이아웃 4종 중 랜덤"]
    K1 --> K2["Text-Shadow + Stroke 적용<br/>이미지 중앙 정렬"]

    K2 --> L["커버 썸네일 박스에<br/>결과물 즉시 렌더링"]

    L --> M["버튼명 변경:<br/>[✦ AI 커버 제거]"]

    M --> N([AI 커버 적용 완료])
```

## AI 커버 제거 플로우

```mermaid
flowchart TD
    A["[✦ AI 커버 제거] 버튼 탭"] --> B[원본 이미지 상태로 복구]
    B --> C["버튼명 변경:<br/>[✦ AI 커버 생성]"]
    C --> D([원본 커버로 복귀 완료])
```

## 이미지 미업로드 시 대체 플로우

```mermaid
flowchart TD
    A{셀러가 커버 이미지를<br/>직접 업로드했는가?} -->|Yes| B[업로드된 이미지 사용]
    A -->|No| C{판매 상품이<br/>등록되어 있는가?}
    C -->|Yes| D["판매 상품 첫 번째의<br/>제품 이미지를 썸네일로 불러오기"]
    C -->|No| E["AI 커버 생성 불가<br/>토스트 메시지 노출"]

    B --> F["[✦ AI 커버 생성] 진행"]
    D --> F
```

## 툴팁 정보 플로우

```mermaid
flowchart TD
    A["ℹ️ 툴팁 아이콘 탭"] --> B["Bottom-up 팝업 노출"]
    B --> C["타이틀: AI 방송커버가 뭐예요?"]
    C --> D["컨텐츠: AI 커버 적용 샘플 이미지 3장"]
    D --> E["서브 텍스트:<br/>방송 제목과 커버 이미지를 기반으로<br/>텍스트 오버레이를 자동 생성합니다."]
    E --> F["팝업 닫기"]
    F --> G([메인 화면 복귀])
```

## Phase 2: 데이터 변경 시 동기화 플로우 (권장안 - B안)

```mermaid
flowchart TD
    A([AI 커버 적용 상태]) --> B{"셀러가 방송 정보를 변경<br/>(제목/상품/혜택)"}
    B -->|변경 감지| C["기존 AI 커버 자동 해제<br/>(원본 이미지로 롤백)"]
    C --> D["토스트 메시지 노출:<br/>'방송 정보가 변경되어<br/>AI 커버가 초기화되었습니다.<br/>다시 생성해주세요.'"]
    D --> E["[✦ AI 커버 생성] 버튼 노출"]
    E --> F{셀러가 다시<br/>AI 커버 생성?}
    F -->|Yes| G["[✦ AI 커버 생성] 탭 → 재생성"]
    F -->|No| H([원본 이미지로 방송 진행])
```

## 전체 통합 플로우 요약

```
[셀러 앱 진입]
      │
      ▼
[라이브 방송 시작]
      │
      ▼
[방송 설정 메인 화면]
      │
      ├──→ 방송 제목 입력 ──────────────────┐
      ├──→ 커버 이미지 업로드 ────────────────┤
      ├──→ 판매 상품 선택 (옵션) ─────────────┤
      ├──→ 강조 혜택 입력 (옵션) ─────────────┤
      │                                      │
      │    ┌───────────────────────────────────┘
      │    ▼
      ├──→ [✦ AI 커버 생성] 버튼 탭
      │         │
      │         ├── 조건 미충족 → 토스트 알림
      │         │
      │         └── 조건 충족
      │              │
      │              ▼
      │         [텍스트 추출 + 렌더링]
      │              │
      │              ▼
      │         [커버에 즉시 반영]
      │              │
      │              ▼
      │         [✦ AI 커버 제거] 버튼으로 전환
      │              │
      │              ├── 제거 탭 → 원본 복구
      │              └── 유지 → 방송 시작
      │
      └──→ [ℹ️ 툴팁] → AI 방송커버 설명 팝업
```
