# 경력 섹션 회사 로고 첨부 가이드

`경력 요약 · 8년+` 타임라인의 회사명 앞 아이콘 자리입니다.
아래 파일명으로 이미지를 이 폴더(`portfolio/assets/logos/`)에 넣으면 **자동으로** 로고가 표시됩니다.
파일이 없으면 큰 이니셜 칩(G·캐·iP·다·엔)이 임시로 보입니다. (`index.html` 수정 불필요)

## 파일명 (붙여준 순서)
| 순서 | 회사 | 파일명 |
|---|---|---|
| 1 | 그립컴퍼니 (Grip) | `grip.png` |
| 2 | 넛지헬스케어 (캐시워크) | `cashwalk.png` |
| 3 | 아이픽셀 (iPIXEL) | `ipixel.png` |
| 4 | 다노 (DANO) | `dano.png` |
| 5 | 엔핏 (현 클래스유) | `classu.png` |

## 규격
- 표시 크기: **44×44px** (회사명+직책 2줄 높이에 맞춤). 흰 배경 + 얇은 테두리 + 둥근 모서리 프레임에 `contain`으로 들어감.
- 권장: **정사각형 PNG**(투명 배경이면 더 깔끔). 가로형 워드마크(CLASSU 등)도 가운데 정렬되어 잘림 없이 표시됨.
- `.png` 외 `.jpg`/`.svg`도 되지만, 파일명 확장자를 쓰면 `index.html`의 `src`도 같이 바꿔야 함(기본은 `.png`).
- 꽉 찬 느낌을 원하면 `index.html`의 `.co-logo`에서 `object-fit:contain`→`cover`, `padding` 제거로 변경 가능.
