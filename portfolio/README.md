## 구성 (단일 스크롤 + A4 인쇄)

이그나이트 톤(네이비+블루+골드 단일 컬러)으로 통일한 **단일 페이지** 포트폴리오:

1. Hero — 핵심 포지셔닝
2. At a glance — 핵심 지표 6종
3. About Me — 일하는 방식 + 강점/도메인
4. Career — 경력 타임라인 (5사)
5. Impact Snapshot — 숫자로 보는 임팩트
6. Case Studies — 대표 9건 (배경→접근→구조→화면→결과→학습)
7. Skills & Tools — 도메인·역량·툴·AI
8. Philosophy — 4가지 원칙
9. Contact

### 특징
- **고정 네비 + 진행바 + 스크롤 스파이** — 긴 문서 이동성
- **카운트업 / 스크롤 리빌** — 단, `prefers-reduced-motion` 및 인쇄 시 비활성
- **실제 화면 첨부 슬롯** — `assets/screens/`에 규칙대로 이미지만 넣으면 자동 반영 (가이드: `assets/screens/README.md`)
- **A4 인쇄 최적화** — 주요 섹션·케이스 단위 페이지 분할
